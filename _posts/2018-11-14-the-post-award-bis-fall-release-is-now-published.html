--- layout: post title: The Post-Award BIS Fall Release is now published date: 2018-11-14 00:00:00 ---

The Fall 2018 release of the PEPPOL Post-Award artefacts (BIS documents and validation artefacts) has now been published.

 

The release package and its release notes can be downloaded at:

https://peppol.eu/downloads/post-award/

 

This release from OpenPEPPOL includes changes to the mandatory validation artefacts for the Invoice Response 63A .

 

The BIS2 specifications in this release are mandatory in use from November 15th 2018 12:00 CET.

 

Please prepare implementation and support for the release as soon as possible to be ready for the mandatory deadline.

 

In addition, this release contains a revision to the BIS3 Billing specification to a version 3.0.2.
The BIS3 Billing is optional and is scheduled to become mandatory on April 18th. 2019.

 

Validation artefacts are the tools that help parties and service providers to comply with the PEPPOL Transport Infrastructure Agreements (TIA) as they help to ensure that the contents of the exchanged document instances in the PEPPOL network are valid both from a syntactical and semantic point of view. These business rules check for the correctness of the coded values, co-occurrence constraints or calculations. It is therefore necessary to use them before submitting documents to the network, in order to ensure the overall quality of exchanged electronic documents, and thus the overall performance of the network.
