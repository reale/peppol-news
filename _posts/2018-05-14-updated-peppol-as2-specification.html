--- layout: post title: Updated PEPPOL AS2 specification date: 2018-05-14 00:00:00 ---

Version 1.01 of the PEPPOL AS2 specification has been released.

 

The change is implemented in order to improve security in the PEPPOL network and relates to which TLS version to be used by Access Point providers in the network.

 

Service providers are mandated to comply with the specification beginning on June 1 2018.

 

Changelog

  • Line 292 now says “It MUSTuse TLS2 as described in RFC 5246.“
  • Previous version said “It SHOULDuse TLS2 where possible as described in RFC 5246.“

 

The specification can be downloaded from https://github.com/OpenPEPPOL/documentation/raw/master/TransportInfrastructure/

 

Please send any change requests to rfc@peppol.eu.
