--- layout: post title: Fall release 2019 date: 2019-11-04 00:00:00 ---

 

The Fall 2019 release of the Peppol Post-Award artefacts (BIS documents and validation artefacts) has been published. This release from Peppol includes changes to the mandatory validation artefacts for all published Peppol BIS 3.0 documents.

 

The release package and its release notes can be downloaded at: https://peppol.eu/downloads/post-award/

 

The BIS 3.0 specifications in this release are mandatory from November 15th, 2019 12:00 CEST. Please prepare implementation and support for this release as soon as possible to be ready for the mandatory deadline.
