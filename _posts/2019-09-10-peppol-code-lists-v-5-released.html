--- layout: post title: PEPPOL Code Lists v.5 released date: 2019-09-10 00:00:00 ---

The OpenPEPPOL eDelivery Change Management Board (EDEC-CMB) is happy to announce the release of PEPPOL Code Lists v.5 for document types, participant identifier schemes, processes and transport profiles. These are valid immediately and can be found at:

https://docs.peppol.eu/edelivery/codelists/

 

A list of all changes compared to the previous version can be found at:

https://docs.peppol.eu/edelivery/codelists/changelog.html

 

On behalf of OpenPEPPOL and the eDelivery CMB, I would like to thank everyone involved for their efforts.
