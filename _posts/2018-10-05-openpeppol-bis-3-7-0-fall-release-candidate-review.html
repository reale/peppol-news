The 2018 Fall release, version 3.7.0, of the PEPPOL Post-Award artefacts (BIS documents and validation artefacts) is now available for review.

 

Deadline for submitting comments is end of day Friday, October 26th.

 

Please send your comments by email to georg@midran.is  by using the Excel comment template.

 

Release notes for 3.7.0 release candidate can be downloaded from here.

 

The 3.7.0 Fall release candidate can be downloaded for review as follows:

 

BIS3 Billing Package 

BIS63A Invoice Response

 

Key points of this release

 

The Fall release only affects two BIS specifications, Invoice Response BIS63A and Billing BIS3.

  •  Changes to one rule and to test files in Invoice Response BIS63A.
  •  Changes to documentation, rules and test files for Billing BIS3.

 

Time plan:

 

26th of October 2018: Deadline for comments.

9th of November 2018: Published as released.

15th of November 2018 12:00 CEST: Mandatory use.

 

Note:

The Open PEPPOL BIS 3.0 Upgrade Work Group is in process. The work group will deliver an upgrade of all Peppol BIS specifications, other than the BIS3 billing, aligning them to the BIS3 Billing specification and thus with the data model, the UBL syntax binding, validations and code lists of the European eInvoice EN16931. This upgrade is planned to be distributed for review next week (week 41) aiming for publication as optional specifications in November and to be adopted as mandatory OpenPEPPOL BIS specification in 2019, replacing the current specifications. The BIS3 upgrade is a non-backwards compatible major release.
